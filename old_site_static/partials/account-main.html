EveKit Account Synchronization

EveKit account synchronization will download character and corporation data on your behalf from the EVE XML API endpoints. We keep an unlimited history of your data, so you can view your data as it appeared any time up to the time you added an account for synchronization. You can also download the latest version of your data in bulk form as a collection of CSV files.

Getting Started:

1 Sign in using one of the third party providers listed in the upper right corner. At EveKit, we don't maintain login credentials (who wants that headache?). We only pull basic identification via OAuth from the third party you use for sign in.
2 Once you're logged in, select "Account Sync -> Create Sync Account" from the menu.
3 On the "Add Account" page, you'll need to:
  • Provide an EVE Online API key and vCode to use for synchronization.
  • Select whether to synchronize a character or corporation.
  • Select which character to use for synchronization.
  • Select whether you want to synchronize automatically or manually.
4 That's it! To access your data online, create a data access key by selecting "Data Access Keys -> Data Access Keys for <your_account>". If you prefer offline access, you can click "Download Latest Snapshot" from the account list (we'll download your data as a zip of CSV files). You can find more information about accounts and access keys on the Account FAQ.
