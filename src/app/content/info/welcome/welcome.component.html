Welcome to EveKit!

EveKit is a web-based toolkit for building applications based on the EVE Online third party APIs. EveKit's main function is to keep up to date copies of your character and/or corporation data stored on the web. EveKit provides convenient REST APIs so you can access your data from your own applications. A list of key features is given below.

EveKit also serves as a data collection and access service for many other EVE Online data sets, including the Static Data Export and EVE Online market data. Most of these data sources have convenient REST APIs, as well as access to historical data in bulk form.

Key Features:

build EveKit services are web-ready. Every API is annotated with Swagger markup making it easy to work directly with REST services (returning JSON), or generate a client in the language of your choice (using free tools like the Swagger Editor).
sync EveKit account synchronization can download for both characters and corporations, at least as frequently as the EVE API cache timers allow.
history Account synchronization keeps a complete history of everything. Want to know what your assets looked like a month ago? No problem. The account model APIs allow you to access your data at a date you specify (defaulting to the latest live data).
label Account data we store on your behalf can be tagged with persistent meta data. This makes it easy to store custom data you might need for certain applications right next to your regular account data.
perm_media EveKit provides interfaces to key reference data like the EVE Static Data Export and EVE Market Data. We always provide access to the last two SDE dumps. We scrape market data continuously (at least as frequently as the cache timers allow), and generate daily dump files for download.
cloud_download Any account data we store on your behalf is always available to you in bulk form (as CSV files). We recognize that not everyone will want to use our APIs to access their data, so we generate daily snapshots which you can access from our account synchronization services. We don't trap your data behind our APIs, you can get a complete snapshot any time you like.
