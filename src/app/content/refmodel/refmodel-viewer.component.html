Reference Data View

EveKit captures regular snapshots of some EVE Online reference data. This data can be accessed using a REST API which also includes an OpenAPI specification. You can generate a client in your favorite language using tools like Swagger CodeGen or the Swagger Editor and the specification URL shown below.

Note that EveKit provides access to both current and historical data. Thus, while the same data can be accessed directly from the ESI, EveKit can also serve historic views.

You can browse the Reference Data API in the Swagger UI below. The Reference Data API uses a special query language which allows filtering fields by value, by range, or SQL-like wildcards on string-valued fields. See our GitHub project for a complete description of the query language.
