Static Data Export View

EveKit provides a REST API for accessing the last two Static Data Export (SDE) releases from CCP. EveKit also exposes an OpenAPI specification for each release. You can generate a client in your favorite language using tools like Swagger CodeGen or the Swagger Editor and the specification URL shown below.

You can browse the SDE API in the Swagger UI below. Use the menu on the right to select the appropriate SDE release.

The SDE API uses a special query language which allows filtering fields by value, by range, or SQL-like wildcards on string-valued fields. See our GitHub project for a complete description of the query language.
